
# Portfolio Starter (React + Vite + TS + Tailwind + Router)

Fast scaffold to push a minimal site live tonight; adapt the DappU template later.

## Quickstart
```bash
npm i
npm run dev
```

## Build & Preview
```bash
npm run build
npm run preview
```

## Pages
- `/` Home
- `/proofmint`
- `/projects`
- `/about`
- `*` 404
